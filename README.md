# TTracker
## What is ttracker
Simple CLI to track your work hours

## Build
### Dependencies
* Golang 1.18

### Instructions
```shell
git clone https://github.com/FritzJo/ttracker.git
cd ttracker
go build *.go
```

## Features
* Clock-in / Clock-out
* CSV based storage
* One file for each year
* Summary of hours worked overtime
